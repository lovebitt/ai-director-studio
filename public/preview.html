📱 访问方式

方案一：本地开发服务器

在服务器上运行开发模式

# 进入网站目录
cd /root/.openclaw/workspace/website

# 安装依赖
npm install

# 启动开发服务器
npm run dev

访问地址: http://localhost:3000

方案二：静态文件预览

直接查看HTML文件内容

# 查看首页源代码
cat src/pages/index.jsx

# 查看组件代码
cat src/components/HeroSection.jsx

# 查看样式文件
cat src/styles/globals.css

方案三：公网部署（推荐）

部署到Vercel或GitHub Pages

# 推送到GitHub仓库
git init
git add .
git commit -m "AI导演网站"
git remote add origin https://github.com/用户名/仓库名.git
git push -u origin main

然后访问Vercel导入部署

🚀 立即部署 📁 查看代码
